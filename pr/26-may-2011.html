PICSHAPASS ADDS STYLE TO SECURITY
-- Android Password Generating Utility Ships --

SAN ANTONIO, TX (26 May 2011) - Today OURBUNNY released the initial versions of PicshaPass and PicshaPass Free on the Android Market(tm). Available for $2 or for free if you can put up with a splash screen. PicshaPass is an Android(tm) utility that generates secure passwords for authentication with computer systems. The easy to use interface along with an innovative method of generating passwords sets PicshaPass apart from the competition. Photos taken with your phone's camera are transformed into a stream of random bits using a secure hashing algorithm known as SHA-256. You decide where to aim the camera, snap the picture, and PicshaPass creates the password - adding a fun twist to an otherwise frustrating task.

Reusing the same password for multiple websites is a dangerous practice that creates multiple points of failure. If one website does not properly store login credentials and is attacked, then all accounts using that password are compromised. You can mitigate these risks by using a unique password for each website. Additionally, long and complex passwords should be used to make it difficult for an attacker to simply guess the correct password.

PicshaPass allows the user to select lowercase, uppercase, numbers, or special characters and choose a password length between 1 and 30. Another interesting feature is the ability to generate a password from pictures already stored in the phone's gallery, allowing you to use a picture of your dog to produce your email password. More information on PicshaPass and OURBUNNY can be found at www.ourbunny.com.
